# DroidExplainAI

## Overview

DroidExplainAI is a research-oriented project that combines Android malware detection, machine learning, explainable AI, and generative AI.

The objective is to classify Android applications and generate human-readable explanations for security analysts.

## Research Inspiration

This project is inspired by my published research on:

- Android Malware Detection
- Opcode N-gram Analysis
- System Call Analysis
- Feature Selection
- Ensemble Learning

## Planned Features

- Android malware classification
- Explainable AI
- Generative AI explanations
- Malware family analysis
